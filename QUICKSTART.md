# Quick Start Guide

## Installation & Run

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`

## First Steps

1. **Choose Your Class**
   - **Vibe Surfer**: Perfect for beginners - focus on learning concepts
   - **Co-Pilot**: For intermediate developers - hybrid approach
   - **10x Architect**: For experts - deep technical challenges

2. **Select Platform & Stack**
   - Choose Web or Mobile
   - Pick your tech stack (React+Node, Flutter+Firebase, or Next.js+Prisma)

3. **Start Playing**
   - Click "Start Quest" to begin
   - Navigate through the 4 phases of development
   - Complete tasks to earn XP and progress

## Game Controls

- **Sanity (HP)**: Keep it above 0 or you'll fail
- **Coffee (Mana)**: Regenerates automatically, needed for actions
- **XP**: Earn by completing tasks, level up to unlock phases

## Challenge Types

- **Prompt Battle**: Choose the best prompt from 3 options
- **Code Review**: Find bugs in generated code
- **Hallucination Hunter**: Identify real libraries from fake ones
- **Console Warfare**: Fix terminal errors
- **CSS Clash**: Fix layout issues with Tailwind
- **Schema Panic**: Map JSON to TypeScript interfaces
- **CI/CD Gauntlet**: Fix deployment pipeline issues

## Tips

- Write detailed prompts (50+ characters) for better AI results
- Manage your Coffee wisely - some actions cost more
- Read the Senior Dev tips for guidance
- Complete tasks in order to unlock new phases
